# Matched Tumor-vs-Normal Differential Expression in Colorectal Cancer Using DESeq2

## Goal
This project analyzes public colorectal cancer RNA-seq data to compare tumor tissue with matched normal tissue.

## What I will do
- download public expression data
- organize metadata
- clean and prepare the dataset
- analyze tumor vs normal differences
- create figures
- summarize biological findings

## Tools
- R
- DESeq2
- GitHub
- VS Code

## Folder structure
- data_raw: original downloaded files
- data_processed: cleaned files
- metadata: sample information
- scripts: analysis code
- figures: plots and images
- results: tables and outputs
- notes: project notes
- report: final written summary
